# Simulacao Tres Corpos

Simulacoes numericas do problema dos tres corpos usando Python. O projeto resolve as equacoes diferenciais do sistema gravitacional e exibe trajetorias/animações com Matplotlib.

## Tecnologias

- Python
- NumPy
- SciPy
- Matplotlib

## Como executar

```bash
pip install numpy scipy matplotlib
python teoria_v3.py
```

Tambem existem versoes anteriores em `teoria_3_corpos.py` e `teoria_3_corpos_v2.py`.

## Aprendizados

- Integracao numerica com `solve_ivp`.
- Modelagem de forcas gravitacionais.
- Visualizacao 3D e animacao de trajetorias.

## Status

Projeto experimental de estudo em fisica computacional.
